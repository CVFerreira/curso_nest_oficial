
## Description

This project was developed during the official course of NestJS <a>https://courses.nestjs.com/
To run this project it is necessary to have Docker installed on the computer as well as Nest

## Installation

```bash
$ npm install
```

## Running the app

```bash

# watch mode
$ npm run start:dev
$ docker-compose up -d
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
